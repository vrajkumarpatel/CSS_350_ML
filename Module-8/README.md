# Module 8: Time Series Analysis

## Overview
Module 8 deals with Time Series Data, where the order of data points is crucial. We analyze trends, seasonality, and patterns over time.

## Key Topics
- **Time Series Decomposition**: Breaking down time series into trend, seasonal, and residual components.
- **Forecasting**: Predicting future values based on past history.
- **Datasets**: Analyzing diverse datasets like weather, finance, and travel.

## Files
- `module_8_assignment.ipynb`: Notebook containing the time series analysis code.
- `AirPassengers.csv`: Airline passengers data.
- `daily_min_temps.csv`: Temperature data.
- `sp500_prices.csv`: Stock market data.
