# Module 4: Regression Analysis

## Overview
Module 4 focuses on Regression, a supervised learning technique used for predicting continuous numerical values.

## Key Topics
- **Predictive Modeling**: Building models to predict housing prices.
- **Data Preprocessing**: Cleaning and preparing data for regression analysis.
- **Model Evaluation**: Using metrics like RMSE and R-squared to evaluate performance.

## Files
- `california_housing_prices.ipynb`: Notebook containing the regression analysis on California housing data.
- `Report.md`: A summary of the findings and model performance.
- `californiahousingprices.csv`: The dataset used for the project.
