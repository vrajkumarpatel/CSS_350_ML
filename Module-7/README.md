# Module 7: Conversational AI

## Overview
This module explores the field of Natural Language Processing (NLP) and the creation of Conversational Agents (Chatbots).

## Key Topics
- **NLP Basics**: Understanding how computers process human language.
- **Bot Architecture**: Designing simple conversational flows.
- **Interaction Analysis**: Analyzing the performance and capabilities of a bot.

## Files
- `conversational_bot.py`: Python script implementing the logic for the conversational bot (Erica).
- `Exploring and Analyzing a Conversational Bot (Erica).docx`: Documentation and analysis of the bot's behavior.
