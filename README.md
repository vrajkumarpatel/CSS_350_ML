# CSS 350: Machine Learning

Welcome to the repository for **CSS 350: Machine Learning**. This repository serves as a comprehensive collection of coursework, assignments, and projects completed throughout the course. It covers a wide range of fundamental and advanced machine learning concepts, practical implementations, and ethical considerations in AI.

## Repository Structure

The course material is organized into modules, each focusing on a specific topic within machine learning:

- **[Module 1: Introduction](./Module-1)**
  - Initial exploration and essays on the impact and scope of machine learning.

- **[Module 2: Supervised Learning](./Module-2)**
  - Fundamentals of supervised learning.
  - Implementation of optimization algorithms.
  - Analysis of real-world datasets (e.g., Seattle Weather).

- **[Module 3: Fairness in Machine Learning](./Module-3)**
  - Critical analysis of bias and fairness in AI models.
  - Case studies using the Adult dataset.
  - Techniques for mitigating bias and ensuring equitable model performance.

- **[Module 4: Regression Analysis](./Module-4)**
  - Predictive modeling using regression techniques.
  - In-depth analysis of the California Housing dataset.

- **[Module 5: Classification](./Module-5)**
  - Building and evaluating classification models.
  - Work with multi-class classification problems (e.g., Cuisines dataset).

- **[Module 6: Clustering](./Module-6)**
  - Unsupervised learning techniques.
  - Clustering analysis and Principal Component Analysis (PCA) on music data.

- **[Module 7: Conversational AI](./Module-7)**
  - Exploration of Natural Language Processing (NLP) concepts.
  - Analysis and implementation of conversational agents.

- **[Module 8: Time Series Analysis](./Module-8)**
  - Working with temporal data.
  - Analysis of datasets like Air Passengers, Daily Temperatures, and S&P 500 prices.

- **[Module 9: Reinforcement Learning](./Module-9)**
  - Introduction to reinforcement learning agents and environments.
  - Simulation projects involving grid worlds and simple agents.

## Technologies Used

- **Python**: Primary programming language.
- **Jupyter Notebooks**: For interactive coding and data analysis.
- **Scikit-Learn**: For machine learning models and algorithms.
- **Pandas & NumPy**: For data manipulation and numerical computing.
- **Matplotlib & Seaborn**: For data visualization.

## Getting Started

To explore the contents of this repository:

1.  Clone the repository:
    ```bash
    git clone https://github.com/vrajkumarpatel/CSS_350_ML.git
    ```
2.  Navigate to the specific module directory to view the notebooks and reports.
3.  Ensure you have the necessary Python dependencies installed to run the notebooks.
